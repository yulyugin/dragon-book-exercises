# 3.1 The Role of the Lexical Analyzer

## Exercise 3.1.1

```

    /* returns x-squared, but never more than 100 */
    return (x<=-10.0||x>=10.0)?100:x*x;
}
```

guide. Which lexemes should get associated lexical values? What should
those values be?
