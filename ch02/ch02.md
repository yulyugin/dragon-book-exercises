## 2.11
Construct recursive-descent parsers for the grammars in Exercise 2.2
(a), (b), (c).

```
// a) S -> 0 S 1 | 0 1
// Same thing:
// S -> 0 S1 1
// S1 -> 0 1 | epsilon

void match(const char token) {
    if (lookahead == token)
        lookahead = token;
    else
        error();
}

void S() {
    if (lookahead == '0') {
        match('0');
        S();
        match('1');
    } else {
        error();
    }
}

void S1() {
    if (lookahead == '0') {
        match('0');
        match('1');
    } else {
        // match(epsilon);
    }
}
```

```
// b) S -> + S S | - S S | a

void S() {
    switch (lookahead) {
    case '+':
        match('+');
        S();
        S();
        break;
    case '-':
        match('-');
        S();
        S();
        break;
    case 'a':
        march('a');
        break;
    default:
        error();
    }
}
```
