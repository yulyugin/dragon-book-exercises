What is the input alphabet of each of the following languages?

a) Pascal

b) C

c) Fortran 77

d) Ada

e) Lisp

### Answer

a) `{a-z A-Z 0-9 - + * / : = , . ; ( ) [ ] { } '}`

b) `{a-z A-Z 0-9 - + * / : = , . ; ( ) [ ] { } ' " ^ & ! ? < > | % ~ _ # \}`

c) `{A-Z 0-9 + - * / ( ) = . , ' : $}`

d) `a-z A-Z 0-9 + - * / ; ( ) . , : = > < " & _ '`

e) `a-z A-Z 0-9 ' " ( ) + - * / = > < & . # ;`
